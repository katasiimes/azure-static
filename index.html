Azure: Kysymyksiä ja vastauksia

  1. Mihin tarkoitukseen Azure AI -palvelut on suunniteltu?
  2. Mikä on Azure Resource Managerin tehtävä?
  3. Mikä ero on Azure Blob Storagella ja File Storagella?
  4. Miten Azure Monitor auttaa sovellusten valvonnassa?
  5. Voiko Azurea käyttää IoT-laitteiden hallintaan?
  1. Azure AI auttaa tekoälyn hyödyntämisessä ilman omaa AI-infraa.
  2. Se hallinnoi resurssien käyttöönottoa ja organisointia.
  3. Blob on objektitallennusta, File tarjoaa SMB-pohjaisen jaetun levytilan.
  4. Monitor tarjoaa lokit, metrikat ja hälytykset sovellusten tilasta.
  5. Kyllä, Azure IoT Hub on suunniteltu juuri tähän tarkoitukseen.
