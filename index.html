Azure: Kysymyksiä ja vastauksia

  1. Mitä Azure tekee?
  2. Voiko Azurella tallentaa tiedostoja?
  3. Toimiiko Azure selaimessa?
  4. Onko Azure Microsoftin tuote?
  5. Voiko Azurea käyttää koodin tallentamiseen?
  1. Azure on pilvipalvelu, jolla voi ajaa sovelluksia ja tallentaa tietoa.
  2. Kyllä, Azurella voi tallentaa tiedostoja eri tallennuspalveluihin.
  3. Kyllä, suurin osa Azuren toiminnoista toimii selaimessa.
  4. Kyllä, Microsoft kehittää ja ylläpitää Azurea.
  5. Kyllä, esimerkiksi GitHubin voi yhdistää Azureen koodin hallintaa varten.
